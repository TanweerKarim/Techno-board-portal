Techno Board Portal

Techno Board Portal is a Portal which enable student to register, Fill examination form, Download Admit card. New features such as online payment for new registration and examination. Now student can view there result online and can even Download a provisional Marksheet till 5th semester. There is a admin panal where the admin can update student details, insert marks scored by students, update marks scored, declare the date of Registration, Exam form,admit card, result. Admin can even add subject if a new syllabus is introduced.
